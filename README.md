# DocScanner-windows
Document scanner for Windows (CamScanner-like) built with Python, OpenCV and CustomTkinter

# DocScanner PDF

## 🇬🇧 English
Document scanner for Windows similar to CamScanner.

Features:
- Automatic document detection
- Manual corner correction
- Black & white cleaning
- Lens correction
- Page flattening
- Multi-page PDF
